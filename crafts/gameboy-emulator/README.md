# Game Boy Emulator from Scratch

Building a Game Boy (DMG) emulator without using any emulation libraries to understand how the hardware works.

## Implementations

- Go (planned)

## Goals

- Emulate the Sharp LR35902 CPU (Z80-like instruction set)
- Implement Memory Management Unit (MMU) and memory map
- Implement Picture Processing Unit (PPU) for tile-based graphics
- Load and run ROM cartridges
- Handle interrupts and timers
